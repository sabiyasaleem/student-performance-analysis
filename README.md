# Student Performance Analysis
##### This project is based on a student performance dataset. The main aim of this project is to analyze how different factors like study time, family background, and lifestyle habits affect students' academic performance.

##### In this project, I have used Python and different data analysis libraries to clean, explore, and visualize the data.

 ### Objectives
##### To understand the dataset and its features
##### To clean and preprocess the data
##### To analyze relationships between different variables
##### To visualize important patterns in student performance
##### To gain insights that can help improve academic results
  ### Technologies Used
 ##### Python 
##### Pandas
##### Matplotlib
##### Seaborn
##### Jupyter Notebook
   ### Dataset Information

#### The dataset contains information about students such as:

##### Personal details (age, gender, address)
##### Family background (parents' education, jobs)
##### Study-related factors (study time, failures, support)
##### Lifestyle factors (free time, going out, alcohol consumption)
##### Academic performance (G1, G2, G3 grades)
  ### Steps Performed
##### Data Loading
##### Data Cleaning
##### Renaming Columns for better understanding
##### Exploratory Data Analysis (EDA)
##### Data Visualization
##### Finding Insights
   ### Sample Analysis

##### Some of the analysis done in this project:

##### Distribution of final grades (G3)
##### Effect of study time on performance
##### Impact of parental education on grades
##### Relationship between failures and final results
  ### Visualizations

##### I have created different types of graphs such as:

##### Histograms
##### Bar charts
##### Count plots


These visualizations help in better understanding of the data.

   ### Key Insights
##### Students with higher study time tend to perform better
##### More past failures generally lead to lower final grades
##### Family and social factors also influence performance
##### Balanced lifestyle plays an important role
